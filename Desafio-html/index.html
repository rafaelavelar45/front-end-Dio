Oque é semântica em html?

semântica seria da sentido as palavras, fazendo um contexto entre as tags em html , trazendo sentido uso das tags.

HTML5 veio com esse proposito deixando de ser algo solto onde se usa tags para formatação , esse nao é mais o objetivo do html

agora o HTML5 tras seu proposito sendo estrutural onde a sentido o uso da tag.

Vamos conhecer abaixo um pouco de algumas tags semanticas.

Tag header

A tag header uma tag semantica que veio junto com HTML5 que se refere eo cabeçalho da minha pagina HTML.

Essa tag faz com oque os motores do google entenda sua estrutura e tras otimização , entendendo que ali e toda parte de cabeçalho da sua pagina.

Tag main

A tag main tras sua semantica dizendo a nossa pagina HTML que ali estro introduzido todo nosso conteudo principal HTML

Entao todo conteudo principal ficara dentro dessa tag , temos que ter ciencia que a tag main so é usada uma por pagina HTML

Não faz sentido ter mais de um par de tags main , se seu HTML tera todo conteudo principal inserido dentro o main

Não fazendo necessario outra tag, isso tras sentido semantico para otimização da nossa pagina HTML.

Tag section

Agora vamos falar um pouco da tag section uma tag onde nos diz cada seção do nosso HTML , essa tag é utilizada quando nosso HTML

tras seções como um campo de uma formulario , a ser preenchido naquelo espaço temos uma seção , dentro da section tras tudo aquilo

Que está ligado ao nosso tema principal , lembre-se use sempre uma tag section quando tiver uma necessidade semantica

Se for so para estilização use tag div, podemos usar mais de um par de tags section no nossa pagina HTML.

Tag footer

A tag basicamente está se referindo ao nosso rodapé, tudo aquilo que fica no final que se refere rodapé, entao podemos usar nossa tag footer

A tag tras o sentido semantico , dizendo que ali está o final da nossa pagina HTML , está localizado o nosso rodapé.

Tag nav

Dizemos que a tag nav , vai está localizado nosso menu, tudo aquilo que tiver menu de navegação, sera nosso nav

geralmente utilizada dentro de um headrer se for o caso, com isso faz sentido para nosso buscador de tela , os mecanismos de busca

Que ali está o menu de navegação do nossa pagina HTML.
